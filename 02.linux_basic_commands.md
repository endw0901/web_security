# Linux command

```
pwd

// lsのhelpを表示
man ls

// Tabで自動補完 (DownloadsをDo + Tabで補完)
cd Do + Tab

// インストール可能なパッケージの「一覧」を更新する。
// 実際のパッケージのインストール、アップグレードなどはおこなわない。
apt-get update

// ダウンロード＆インストール
apt-get install terminator
```

## terminator

- 通常の terminal 機能と同じで、追加機能あり
- 右クリック => Split Horizontally で画面分割
